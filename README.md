# Programa-com-acessibilidade
Programa feito a partir de atividade proposta, com acessibilidade e usabilidade

# Proposta do projeto
Programa desenvolvido para facilitar o registro de medicações e pressão de pessoas que usam o aparelho de aferimento de pressão e necessitam de registros para o histórico de medicações. Apenas uma única tela onde o usuário deve informar a data,a hora, a pressão sistólica, a pressão diastólica e se ocorreu alguma situação de estresse no momento que tomou a medicação. Os dados serão colocados em um arquivo csv.

# Foco do projeto
Como é voltado para pessoas idosas, o programa possui aspectos de usabilidade e acessibilidade, como:
* Caracteres maiores;
* Descrições acessíveis e *tooltips* nos componentes presentes na tela;
* Associação de rótulos aos componentes correspondentes;
* Ordem sequencial de foco nos componentes de tela;
* Atalhos por teclado;
* Mensagens de erros amigáveis e simples de entender;
  
# Ferramentas e idéias
Feito pela IDE NetBeans, utilizando a linguagem Java e interface gráfica.
